# Custom-Macropad
A custom designed macropad for personal use! 

* 4 customizbale keys
* 128x32 OLED Display
* EC11 Rotary Encoder
* 7 WS2812B RGB LEDs. Four for backlighting and three for underglow
* SEEED XIAO Rp2040 MCU

QMK Firmware
* LED pulses under kwy when pressed
* OLED graphics
* Controls the macros
Incorporates four keys, a rotary encoder, one 1x0.91 inch OLED display, and seven RGB neopixels for underglow and backlighting. The case is built sandwhich style, with a built in adjustable kick stand to prop up the macropad to a tilt for better ergonmics. The macros currently include functions for desk switching, toggling touchscreen on/off, pulling up To Do list, switching to international keyboard, MLA formatting, adjusting screen birghtness and audio volume, and for switching between layers.
